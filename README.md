# Control-Rakatuy

Repository Control Rakata

## Struktur Folder

* **.git/**: Git configuration (Do not modify)
* **Hardware/**: KiCad PCB layout and schematic files. Nanti buat folder baru aja per project
* **Firmware/**: Firmware yang dipake
* **Docs/**: Documentation
* **Exports/**: (Gerbers, BOM, and PDF schematics)
* **testing/**: ngetes repo

---

### Cloning to a New Machine
To pull this repository into a new local folder:
```bash
git clone https://github.com/KoJZz/Control-Rakatuy

```
### Upload konten baru
```bash
git pull origin main # pull updatean terbaru dari repo
git add . # add file terbaru yang udah dibuat
git commit -m "Update: [Component Name] - Added power stage layout" # commiting
git push origin main # push ke repo

```
